텍스트 색상 강조

텍스트의 색상을 통하여 중요도에 대한 수준을 표현할 수 있습니다.

회색으로 표시되는 비활성 메시지 입니다.

밝은 파란색으로 표시되는 중요 메시지 입니다.

초록색으로 표시되는 성공 메시지 입니다.

짙은 파란색으로 표시되는 정보 메시지 입니다.

노란색으로 표시되는 경고 메시지 입니다.

빨간색으로 표시되는 위험 메시지 입니다.

색상 강조 관련 클래스 들은 다른 inline-level 요소에 적용하여 한문단 안에서 여러 색상 을 표현할 수 있습니다.
